# 使用指南 (Documentation)

---

## ⚙️ 配置 (Configuration)

### 1. 获取爱快 API 令牌
1. 登录爱快 4.0+ 管理后台。
2. 前往 **设备设置 -> 登录管理 -> 个人 API 令牌**。
3. 点击 **添加**，生成一个新的令牌。
4. **重要**：请确保为令牌勾选了“负载监控”、“终端监控”、“接口监控”、“系统升级”、“系统备份”、“安全管理”等相关权限。
5. 获取令牌字符串（Token）。

### 2. 在 HA 中添加集成
1. 前往 HA **设置 -> 设备与服务 -> 添加集成**。
2. 搜索并选择 **iKuai Connect**。
3. 输入爱快地址（需包含协议，如 `https://10.10.10.1`）及刚才生成的 **Token**。
4. 点击提交即可完成。

## 💡 自动化支持 (Actions/Services)

本集成提供了强大的服务接口，可在自动化中直接调用：

| 服务名称 | 描述 |
| :--- | :--- |
| `get_traffic_ranking` | 获取当日终端流量消耗排名 (MB)。 |
| `get_protocol_stats` | 查询特定终端的协议构成 (如视频、游戏占比)。 |
| `add_mac_rule` | 动态添加 MAC 黑/白名单规则（支持过期时间）。 |
| `delete_mac_rule` | 彻底删除指定的 MAC 规则。 |
| `reboot` | 创建一次性计划实现路由器 1 分钟内重启。 |
